# ちょこみんず
スースー不足のあなたへ
<br>チョコミント専門の記録&共有アプリ
<br>〜あなたにぴったりの「美味しい」と「爽快」を探そう！〜

サービスURL : https://chocomins.vercel.app/

# ■サービス概要
・レビューの記録
<br>・メーカー・店舗の登録
<br>・Xでシェア機能

# ■開発背景
私の身近にチョコミント好きの方がいます。
毎年、チョコミント商品が発売される時期になると、新作の食べ比べをしたり、東北の方までローカルなチョコミント商品を求めて足を運んでいます。
お互いに見つけたチョコミント商品を持ち寄って食べ比べや、店舗に食べに行ったこともありますが、万人受けの商品ではないため、まだまだ情報収集の場が少ないように思いました。

ふらっと出かけた先で偶然見つけたチョコミント商品。
こんなところにもあったのか！という情報をみんなで共有したい。
チョコミント好きのプラットフォームになればと、そんな思いで開発をしました。

# ■ターゲット層
20〜30代の層

# ■ターゲット層の理由
20~30代の層 チョコミントが好きな人が多い世代のため


# ■ユーザー獲得方法
Xでの共有機能、チョコミントやチョコが流行るイベントの時期にSNSを使って宣伝していきます。

# ■サービス利用のイメージ
チョコミントを食べた際に、食べたチョコレート登録し、レビューを記録します。
<br>「ちょこみんず」にいつ、どこで、何を食べたかを記録しておく事で、次回チョコミント商品を選ぶ際の参考になります。
店舗の場所やミント感も記録しておく事で、より効率的にチョコミント商品を探すことが出来ます。
<br>また、レビューは公開されるため他の人のレビューを閲覧することも可能です。
<br>チョコミント商品に迷ってもこの機能を使うことで効率的にチョコミント商品を見つけることが出来ます。




# ■主要な機能
- メーカー/店舗管理機能
- レビュー管理機能
- コメント機能
