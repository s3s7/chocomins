# ちょこみんず
〜チョコミント専門の口コミ・共有アプリ　あなたにぴったりの「美味しい」と「爽快」を探そう！〜

サービスURL : https://meguri-gift.com/


# ■サービス概要
・商品の記録
・レビューの記録
・メーカー・店舗の登録

# ■開発背景
誕生日や特別な日を大切にしています。相手のことを考えながらギフトを選ぶ時間が大好きで、自分のこと以上にこだわってしまうところがあります。
ギフトを用意してから渡すまでのワクワクする時間も大好きです。喜んでもらった時の記録を残したくて開発しました。
- 同じギフトを贈らないように
- どんなギフトが喜ばれたのか記録
- 洋服のサイズ等の記録

「60代 父の日」等ネットで調べればおすすめのプレゼントはたくさん出てくるが、結局毎年同じようなもの、PR商品などが検索で上がってきます。

毎年毎年名前入りのグラスやジョッキが出てきても正直ピンとこなかったです。

しかし職場で話していると自分では思いつかないようなギフトを贈っている人がいました。
これを投稿で見れるようにしたらギフトの幅も広がり、楽しいのではないかと考えました。
そして自分にとっては毎年贈っている商品でも、それが誰かにとっての初めての特別なギフトになるかもしれないと思いました。


-私の身近にチョコミント好きの方がおり、一緒にチョコミントの商品を食べ比べているとそれぞれの味の違いに驚きました。
まずミントの爽快さが商品よって異なります。
チョコの甘さも違います。
それらも含めた全体の味の違いもあります。
このような深い違いがあるのにも関わらず、チョコミント商品に関する口コミをまとめて収集する場所がないと知り、そのようなアプリがあれば効率的に自分好みのチョコミント商品を探すことが出来ると考え、開発に至りました。


# ■ターゲット層
20〜30代の層
20代…結婚祝いや出産祝いを初めて贈ることが多い年代
自分もそうだったがどんなものを贈ればいいか、実際に役に立つもの・喜ばれるものは何なのか分からないことだらけでした。

30代…祖父母・両親・甥姪など様々な世代にギフトを贈ることが多いため
特に義実家となると好みが分からず毎年同じようなものになってしまいます。

# ■ターゲット層の理由
20~30代の層 様々なライフイベントが増え、ギフトを贈る機会が多くなるため。

チョコミントが好きな人が多い世代のため

# ■ユーザー獲得方法
Xでの共有機能、チョコミントやチョコが流行るイベントの時期にSNSを使って宣伝していきます。

# ■サービス利用のイメージ
ギフトを贈るイベントが発生した際に、贈ったギフトを記録します。
「めぐりギフト」にいつ、誰に、何を贈った/貰ったかを記録しておく事で、次回ギフトを選ぶ際の参考になります。
相手の反応や好み、サイズ等も記録しておく事で、より喜んでもらえるギフトを選ぶことが出来ます。
また、自分の記録を公開する機能もあるため他の人が贈ったギフトを閲覧することも可能です。
たとえギフトに迷ってもこの機能を使うことで思いもよらないギフトを見つけることが出来ます。



-
チョコミントを食べた際に、食べたチョコレート登録し、レビューを記録します。
「ちょこみんず」にいつ、どこで、何を食べたかを記録しておく事で、次回チョコミント商品を選ぶ際の参考になります。
店舗の場所やミント感も記録しておく事で、より効率的にチョコミント商品を探すことが出来ます。
また、自分のレビューをが公開されるため他の人のレビューを閲覧することも可能です。
チョコミント商品に迷ってもこの機能を使うことで効率的にチョコミント商品を見つけることが出来ます。

- コメント機能
