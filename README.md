# ちょこみんず
スースー不足のあなたへ
<br>チョコミント専門の記録&共有アプリ
<br>〜あなたにぴったりの「美味しい」と「爽快」を探そう！〜

サービスURL : https://chocomins.vercel.app/

# ■サービス概要
・レビューの記録
<br>・メーカー・店舗の登録
<br>・Xでシェア機能

# ■開発背景
私の身近にチョコミント好きの方がいます。
毎年、チョコミント商品が発売される時期になると、新作の食べ比べをしたり、東北の方までローカルなチョコミント商品を求めて足を運んでいます。
お互いに見つけたチョコミント商品を持ち寄って食べ比べや、店舗に食べに行ったこともありますが、万人受けの商品ではないため、まだまだ情報収集の場が少ないように思いました。

ふらっと出かけた先で偶然見つけたチョコミント商品。
こんなところにもあったのか！という情報をみんなで共有したい。
チョコミント好きのプラットフォームになればと、そんな思いで開発をしました。

また商品を食べ比べているとそれぞれの個性があることに気がつきました。 
癖になる爽快感やチョコミント初心者でも入りやすい商品などそれぞれの違いを記録、共有出来るサービスにしました。

# ■ターゲット層
<br>世代にこだわらずチョコミント好きの方
<br>主に20〜40代の層

■ターゲット層の理由
<br>ネットを積極的に活用している世代のため
<br>20~40代の層 チョコミントが好きな人が多い世代のため

■ユーザー獲得方法
<br>Xでの共有機能、チョコミントやチョコが流行るイベントの時期にSNSを使って宣伝していきます。

■サービス利用のイメージ
<br>チョコミント商品を食べた際に、商品の詳細を記録します。商品名、メーカー、ミント感、チョコ感、感想、購入場所を入力することができ、みんなに公開されます。
<br>みんなの投稿から気になる商品を見つけることが出来ます。




# ■主要な機能
- メーカー/店舗管理機能
- レビュー管理機能
- コメント機能
