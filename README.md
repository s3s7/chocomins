# ちょこみんず
〜チョコミント専門の口コミ・共有アプリ　あなたにぴったりの「美味しい」と「爽快」を探そう！〜


# ■サービス概要
・商品の記録
・レビューの記録
・メーカー・店舗の登録

# ■開発背景
私の身近にチョコミント好きの方がおり、一緒にチョコミントの商品を食べ比べているとそれぞれの味の違いに驚きました。
まずミントの爽快さが商品よって異なります。
チョコの甘さも違います。
それらも含めた全体の味の違いもあります。
このような深い違いがあるのにも関わらず、チョコミント商品に関する口コミをまとめて収集する場所がないと知り、そのようなアプリがあれば効率的に自分好みのチョコミント商品を探すことが出来ると考え、開発に至りました。


# ■ターゲット層
20〜30代の層

# ■ターゲット層の理由
20~30代の層 チョコミントが好きな人が多い世代のため


# ■ユーザー獲得方法
Xでの共有機能、チョコミントやチョコが流行るイベントの時期にSNSを使って宣伝していきます。

# ■サービス利用のイメージ
チョコミントを食べた際に、食べたチョコレート登録し、レビューを記録します。
「ちょこみんず」にいつ、どこで、何を食べたかを記録しておく事で、次回チョコミント商品を選ぶ際の参考になります。
店舗の場所やミント感も記録しておく事で、より効率的にチョコミント商品を探すことが出来ます。
また、レビューは公開されるため他の人のレビューを閲覧することも可能です。
チョコミント商品に迷ってもこの機能を使うことで効率的にチョコミント商品を見つけることが出来ます。




# ■主要な機能
- 商品登録機能
- レビュー機能
- コメント機能
